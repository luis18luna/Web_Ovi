Informacion General

En este OVI se trabajará una temática con sus diferentes componentes, la cual es muy interesante y destacaremos algunos puntos que nos llevara a entenderla totalmente. Este Ovi esta propuesto para el desarrollo de la actividad actual del curso Diseños de Sitios Web. Este Ova esta propuesto para el desarrollo de la actividad actual del curso Diseños de Sitios Web. Que está planteado para el desarrollo grupal y que se trabajaran diferentes aspectos de las herramientas utilizadas para llevar a cabo esta actividad, y así obtener lo que se solicita.

Objetivos

Los Objetivos del OVI son: Explicar detalladamente la temática para una mejor comprensión. Evaluar las capacidades del estudiante para desarrollar una temática. Demostrar la capacidad de usar herramientas para procesar la información de una manera práctica. Exponer diferentes conceptos e ideas que sirvan de fundamento para lo que se está trabajando. Los objetivos del curso son: Hacer que el estudiante supla más su conocimiento con los temas que se trabajan en el curso y así el pueda desarrollar diferentes capacidades que le ayudaran en su formación.

Temáticas

En este apartado se trabajarán diferentes temáticas referentes a las Herramientas ETL, es decir la temática y sus componentes o subtemas del cual se componen, que entre los cuales tenemos. Definición, Función, Extracción, Transformación y carga. Aquí destacaremos toda la información relevante de las herramientas ETL para luego plasmarlas y mostrar una temática que llame la atención de las personas ya que es importante sobre todo para los que trabajamos con sistemas, con esto hacemos que esta temática sirva de ayuda a todas esas personas que la necesiten.
